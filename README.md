# hxenv

Simple tool to set haxelib libraries' versions as defined in ./.hxenv

## Usage

```bash
hxenv --use
```

For additional information, see `hxenv --help`.

## .hxenv structure

```json
{
    "libs": [
        {
            "name": "hxcpp",
            "version": "3.3.49"
        }
    ]
}
```
